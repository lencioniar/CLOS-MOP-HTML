Generic Function CLASS-PRECEDENCE-LIST

Syntax:

class-precedence-list class => precedence-list

Arguments and values:

class -- A class metaobject.

precedence-list -- List of class metaobjects.

Description:

Returns the class precedence list of class. The elements of this list are class metaobjects.

During class finalization finalize-inheritance calls compute-class-precedence-list calls to compute the class precedence list of the class. That value is associated with the class metaobject and is returned by class-precedence-list.

This generic function signals an error if class has not been finalized.

Methods:

class-precedence-list (class standard-class)
class-precedence-list (class funcallable-standard-class)
class-precedence-list (class forward-referenced-class)
class-precedence-list (class built-in-class)
